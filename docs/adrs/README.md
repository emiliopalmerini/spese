# ADR Index

Questo indice raccoglie le Architecture Decision Records per il progetto.

Convenzioni:
- File: `NNNN-titolo-kebab-case.md` (es. `0001-auth-service-account.md`).
- Stato in testa al file: `Proposed | Accepted | Superseded`.
- Template: vedi [`0000-template.md`](./0000-template.md).

Linee guida:
- Cattura il contesto, la decisione e le conseguenze.
- Mantieni i trade-off espliciti. Se una decisione viene superata, crea una nuova ADR che la sostituisce.
